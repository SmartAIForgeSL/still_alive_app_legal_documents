## ğŸ“± About Still Alive

Still Alive is a mobile application designed to provide peace of mind for you and your loved ones. Users can confirm they are alive by checking in regularly through the app. If a user doesn't check in within their configured time interval, the app automatically sends notifications to their designated emergency contacts.

### Key Features:
- Regular check-in reminders
- Customizable check-in intervals
- Emergency contact management
- Automatic email notifications
- Multi-language support
- Secure data storage
